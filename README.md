# MTA San Andreas LUA

This is a VS Code extension to add syntax highlighting for MTA San Andreas LUA to VS Code.

Currently it contains:
- Snippets (From 50p' Sublime Text)
- Syntax Highlighting

It comes in two flavors:
- VSDark
- Monokai
